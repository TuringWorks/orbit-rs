MVCC-Enabled SQL Parser and Executor for Orbit-RS

This module provides a comprehensive SQL engine featuring Multi-Version Concurrency Control (MVCC) for deadlock prevention, high concurrency, and ACID compliance. The engine supports all major ANSI SQL constructs while maintaining compatibility with vector operations and Orbit‚Äôs distributed architecture.

  • Vector Operations: AI/ML workloads with pgvector compatibility
  • Transaction Management: ACID compliance with isolation levels
