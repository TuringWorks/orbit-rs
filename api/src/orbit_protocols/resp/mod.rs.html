1//! RESP (Redis Serialization Protocol) adapter for Orbit
2//!
3//! This module implements RESP2 and RESP3 protocols, allowing Redis clients
4//! to interact with Orbit actors as if they were Redis data structures.
5//!
6//! ## Supported Commands
7//!
8//! ### Actor State Commands
9//! - `GET <actor_id>` - Get actor state
10//! - `SET <actor_id> <value>` - Set actor state
11//! - `DEL <actor_id>` - Delete/deactivate actor
12//! - `EXISTS <actor_id>` - Check if actor exists
13//!
14//! ### Hash Commands (for structured actor state)
15//! - `HGET <actor_id> <field>`
16//! - `HSET <actor_id> <field> <value>`
17//! - `HGETALL <actor_id>`
18//!
19//! ### Pub/Sub (for actor events)
20//! - `PUBLISH <channel> <message>`
21//! - `SUBSCRIBE <channel>`
22//! - `PSUBSCRIBE <pattern>`
23//!
24//! ### List Commands (for actor collections)
25//! - `LPUSH <actor_id> <value>`
26//! - `RPUSH <actor_id> <value>`
27//! - `LRANGE <actor_id> <start> <stop>`
